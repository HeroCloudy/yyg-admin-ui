## 组件 API

### Attributes 属性

| 参数 | 说明 | 类型 | 可选值 | 默认值 |
|  ----  | ----  | ----  | ----  | ----  |
|  ----  | ----  | ----  | ----  | ----  |
| logo | logo路径 | string | - | - |
| expandLogo | 展开时的logo路径 | string | - | 使用logo的值 |
| appName | 应用名称，收缩时会隐藏 | string | - | - |
| isExpand | 是否展开 | boolean | true - 展开；false - 收缩 | true |
