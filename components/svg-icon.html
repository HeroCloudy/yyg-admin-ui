SvgIcon SVG 图标 #

SvgIcon 组件用于显示 SVG 图标，既可以支持项目本地的 SVG 图标，也可以支持在线的 SVG 图标。

基本使用 #

使用在线图标 #

在线图标
显示在线图标，直接将图标的 URL 传给组件的 icon 属性即可。
<template>
  <yyg-svg-icon icon="http://www.yygnb.com/demo/car.svg"></yyg-svg-icon>
</template>

<template>
  <yyg-svg-icon icon="http://www.yygnb.com/demo/car.svg"></yyg-svg-icon>
</template>

使用本地图标 #

使用前需要在工程中做如下配置：

  1. 在工程中安装 vite 插件：
shell
pnpm install vite-plugin-svg-icons -D

pnpm install vite-plugin-svg-icons -D
1
  1. 在 vite.config.ts 文件中配置该插件：
typescript
// ...
import { createSvgIconsPlugin } from 'vite-plugin-svg-icons'
// ...

export default defineConfig({
	// ...
  plugins: [
    // ...
    createSvgIconsPlugin({
      // 要缓存的图标文件夹
      iconDirs: [path.resolve(__dirname, 'src/svg')],
      // 执行 icon name 的格式
      symbolId: 'icon-[name]'
    })
  ]
  // ...
}

本地图标
显示本地图标，将图标名称传给组件的 icon 属性。
<template>
  <yyg-svg-icon icon="car"></yyg-svg-icon>
</template>

</template>

自定义样式 #

通过 className 属性设置图标的样式，但此时 className 对应的类不能放在 scoped 中。

默认样式

本地图标设置className

在线图标设置className

自定义样式
通过 className 属性指定全局样式类
<template>
  <div class="svg-icon-demo-3">
    <div>
      <h4>默认样式</h4>
      <yyg-svg-icon icon="car" />
    </div>
    <div>
      <h4>本地图标设置className</h4>
      <yyg-svg-icon class-name="demo-icon" icon="car" />
    </div>
    <div>
      <h4>在线图标设置className</h4>
      <yyg-svg-icon class-name="demo-icon" icon="http://www.yygnb.com/demo/car.svg" />
    </div>
  </div>
</template>
<style lang="scss">
.svg-icon-demo-3 {
  h4 {
    margin: 10px 0;
  }
  .demo-icon {
    font-size: 50px;
    color: rebeccapurple;
  }
}
</style>
