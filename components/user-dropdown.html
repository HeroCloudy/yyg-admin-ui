UserDropdown 头像下拉 #

点击用户头像显示的下拉选，通常位于界面布局的顶部右侧，如下图所示：

基本使用 #

基本使用
通过设置 avatar 属性显示用户头像
<template>
  <yyg-user-dropdown avatar="https://herocloudy.github.io/yyg-admin-ui/logo.png"
                     @avatar-click="onAvatarClick"/>
</template>

<script lang="ts" setup>
import { ElMessage } from 'element-plus'

const onAvatarClick = () => {
  ElMessage.info('点击头像')
}
</script>

基本使用
使用 dropdownItems 属性指定点击头像时的下拉选项
<template>
  <yyg-user-dropdown avatar="https://herocloudy.github.io/yyg-admin-ui/logo.png"
                     :dropdown-items="dropdownItems"
  ></yyg-user-dropdown>
</template>

<script lang="ts" setup>
import { UserDropDownList } from '../../../packages/user-dropdown/src/types'
import { ElMessage } from 'element-plus'

const dropdownItems: UserDropDownList = [
  { title: '修改资料', click: () => { onItemClick('修改资料') } },
  { title: '个人中心', click: () => { onItemClick('个人中心') } },
  { title: '退出登录', click: () => { onItemClick('退出登录') }, isDivided: true }
]

const onItemClick = (msg: string) => {
  ElMessage.success(`点击 ${msg}`)
}
</script>

</style>

组件 API #

Attributes 属性 #

参数说明类型可选值默认值
avatar用户头像URLstringnull
size用户头像大小number-36
defaultAvatarText没有头像时使用该字符串代替头像string-'User'
dropdownItems下拉选列表，数组每项类型为 UserDropdownItemUserDropdownList-null

UserDropdownItem #

参数说明类型可选值默认值
title下拉选文字string-
click该项点击事件() => void
isDivided是否显示分割线booleanfalse
